## Controls
- ``Right arrow``: Move right
- ``Left arrow``: Move left
- ``Up arrow``: Move up
- ``Down arrow``: Move down
- ``Q``: Quit
- ``SPACE``: Restart

## How to use
You can compile and run this app by yourself OR you can use my [make.sh] script:
#### Build all .class files
```shell script
$ sh make.sh build
```

#### Run the app
```shell script
$ sh make.sh run
```

#### Build JAR file
```shell script
$ sh make.sh build-jar
```

#### Clean
```shell script
$ sh make.sh clean
```
